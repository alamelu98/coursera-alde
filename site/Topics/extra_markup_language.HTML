The version HTML 5 is mostly supported by different browsers and has many useful tags.

DOCTYPE

Doctype is used to say which version of HTML is being used ,"!DOCTYPE HTML" is for HTML 5

COMMENTS IN HTML

Comments in HTML is not reflected as the final output.But seen at the Source code." <!comments...... >

ATTRIBUTES

  1. "ID ATTRIBUTE" is used to uniquely identify an element.It cannot be used by anyone.They are also called as GLOBAL ATTRIBUTES
  2. "CLASS ATTRIBUTE" is used to differentiate between any other elements.It Can be repeatedly used by other elements

GROUPING OR SECTIONING ELEMENTS

  1. "DIV TAG" is used to arrange all the elements one below the another
    Div tag
    Example: Mainly to arrang Block elements
  2. "SPAN TAG" is used to arrange all the elements side by side
    Span tag
    Mainly to arrange inline elements

INLINE AND BLOCK ELEMENTS

BLOCK ELEMENTS

Block elements usually start in a new line.Some common examples are <p>,<h2>,<li>,<ol>.

INLINE ELEMENTS

Inline elements usually continue in the same line.Some common examples are <b>,<q>,<i>,<u>.
