TABLES
Table in HTML are widely used to represent information in the form of grid.The following elements are used for creating Table in HTML.The information in a table id represented in the form of rows.
  • table tag is used to define a table
  • tr tag is used to create rows in table
  • td tag is used to create content in each cell
  • th tag is used to create headings for each row and column using scope="row" or scope="column attribute
  • rowspan attribute is used to gives vertical spaces for each cell
  • colspan attribute is used to give horizontal spaces for each cell
  • thead tag is used to cread a table with long content
  • tbody tag is used to create the content in a long table
  • tfoot tag is used to create footer notes for the long table
