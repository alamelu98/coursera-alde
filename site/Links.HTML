Types of linking the webpages

Every type of webpages are linked using the a tag along with the href attribute

  • Linking to other websites
  • Linking to other websites which are part of same webpages
  • Linking to open the webpage in a new window
  • Linking to same part of the page
  • Linking to specific part of other webpage
  • Link to open a mail

Linking to other websites

Other websites
The webpages can be linked to other webpages by the usage of ABSOLUTE URL.Here,The webpage can be linked by using the domain name of other webpage
Eg:Click on this page to go to wikipedia

Linking to webpages of same file

Same file
The webpages can be linked to other webpages by the usage of RELATIVE URL.Here,The webpage can be linked by using the shorthand name of other webpage.
Eg1:Click on this page to go to lists page
Eg2:Click on this link to go to main page

Opening the webpage in a new window

Open a webpage in New window
The webpages can also be used to open in a new page or in a new window with the help of Target="_blank"
Eg1:Click on me to open in a new window
Eg2:Click on me to open the RELATIVE URL in a new window

Linking to same part of the webpage

Same part of the web page
This type will help you to move to the same part of the webpages with the help of the Id="type1"
Eg1:Click on me to move to top

Linking to new part in other page

Open specific part of another webpage
The webpages can also be used to open a specific part in a new pageId="Type1"
Eg1:Click on me to open "did you know "part in wikipedia

Link to open a mail

Opening a mail id
The webpages can also be used to open and send a mail using mail to
